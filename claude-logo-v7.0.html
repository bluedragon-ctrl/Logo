▸ MOTION

LEFT / LT nturn left n degrees

HOMEreturn to center, heading 0
SETPOS x yjump to position x y
XCOR / YCORcurrent x / y coordinate
HEADINGcurrent heading in degrees
TOWARDS x yangle needed to face x y
DISTANCE x ydistance to point x y

▸ DRAWING
PENUP / PUpen up — stop drawing
PENDOWN / PDpen down — start drawing
PENDOWN? / PD?1 if pen is down, 0 if up
PENUP? / PU?1 if pen is up, 0 if down
CIRCLE rdraw circle, radius r px
DOTfilled dot at turtle position (size = pen width, color = pen color)
DOT x yfilled dot at (x, y) — no turtle movement
SETCOLOR / SC r g bline color (0–255 each)
SETCOLOR / SC namenamed colour — see ▸ COLORS section
SETWIDTH / SW nline width in pixels
LABEL textdraw text at turtle position e.g. LABEL "HELLO
CLEARSCREEN / CSclear screen, reset turtle
CLEARTEXT / CTclear console output
HIDETURTLE / HThide turtle
SHOWTURTLE / STshow turtle
▸ CONTROL
REPEAT n [...]repeat block n times
REPCOUNTcurrent iteration index inside REPEAT (1 to n)
WHILE [cond] [...]loop while condition is true
FOR :v start end [...]count loop (step defaults to ±1)
FOR :v start end step [...]count loop with explicit step
IF cond [...]run block if condition true
IFELSE c [...] [...]if/else branches
STOPexit current procedure early
BREAKexit the innermost loop early (REPEAT / FOR / WHILE)
OUTPUT / OP valreturn value from procedure
WAIT npause for n seconds
SPEED nset animation speed 1 (slow) – 4 (instant)
TO name / ENDdefine a procedure
; commentrest of line is ignored
